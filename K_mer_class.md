# K_mer_class
---
Данный скрипт содержит класс K_mer(*seq*, *position*, *locus*), об аргументе *seq* - можно прочитать в разделе, рассказывающем об атрибутах данного класса,
об аргументах *position* и *locus* рассказывается внутри раздела про метод **recieve(*position*, *locus*)**. Объекты данного класса предназначены для хранения информации о k-мерах найденых в какой-либо последовательности.  
---
## K_mer имеет следующие атрибуты:
1. **seq** - последовательность k-мера  
2. **counter** - счетчик, показывает число вхождений данного k-мера в последовательности  
3. **pos** - список, содержащий в себе координаты нахождения k-мера, представляет из себя список в первом положении которого указано имя последовательности в fasta файле, а во втором положении находится позиция нуклеотида с которого начинается k-мер  
4. **summary** - список содержащий последовательность k-мера и все его вхождения  
---
## K_mer имеет следующие методы:  
1. __recieve(__*position*, *locus*__)__ - получает и сохраняет данные о повторном вхождении k-мера, требует на вход два аргумента:  
'''*position* - номер нуклеотида с которого начинается k-мер  
'''*locus* - имя последовательности или ее номер в fasta файле  
2. **summarize()** - создает и выводит **summary**
